Welcome, and thank you for using Soil Assistant!

Soil Assistant is a great tool for helping you manage your farmland. If you don't know where to start, don't worry â€“ we'll cover all you need to know about using this program in this tutorial.

First, you need an account. If you don't have an account, make sure to click "Sign up". If you already have one, click "Log in".

Fill in all the boxes on the sign up page. Remember your username and password, as they will be needed to log in. After filling all the boxes, press "Create Account".

You will be directed to your account's main menu. To set up your farm, click the "Create new plot" button.

Complete all questions in the survey. Please answer the questions properly. If you have tested for nutrients (recommended), be sure to include amounts of each nutrient in the shown question. If not, leave it blank, and we will decide for you. After you are done, click the "Submit" button.

To edit your profile information, you can press "My account" and make changes.
